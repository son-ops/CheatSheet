# CheatSheet
SQL injection cheatsheet của 5 database phổ biến (MySQL, PostgreSQL, MSSQL Server, Oracle, SQLite).

`Note`:
- Với SQLi nằm ở ORDER BY hoặc là INSERT ta sử dụng subquery - () kết hợp với các kỹ thuật khác loại trừ UNION BASED
- Prepared Statements không thể bind column hay table name
